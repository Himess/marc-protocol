// ARCHITECTURE

Four ERC standards.
One protocol.

ERC-7984 — Confidential Token
Encrypted cUSDC via Zama FHE. Balances and transfers stay encrypted on-chain. 0.1% wrap/unwrap fee. Zaiffer cUSDC migration planned for shared liquidity.
x402 — Agent Payments
HTTP 402 Payment Required standard. Agent auto-pays for APIs with encrypted cUSDC. SDK middleware: one line to add paywall.
ERC-8004 — Agent Identity
Trustless Agents standard. On-chain identity registry, reputation scoring, wallet linking. Agents build trust through verifiable payment history.
ERC-8183 — Agent Commerce (Draft, Feb 2026)
Agentic Commerce Protocol by Virtuals + Ethereum Foundation dAI team. Job escrow with confidential payments. 1% platform fee on completion. Spec evolving — we're building alongside it.
