// ARCHITECTURE

Four ERC standards.
One protocol.

ERC-7984 — Confidential Token
ConfidentialUSDC: wrap USDC → encrypted cUSDC. Balances and transfers stay encrypted via Zama FHE. Fee: 0.1% on wrap/unwrap.
x402 — Agent Payments
HTTP 402 Payment Required standard. Agent auto-pays for APIs with encrypted cUSDC. SDK middleware: one line to add paywall.
ERC-8004 — Agent Identity
Trustless Agents standard. NFT-based identity, on-chain reputation, validation registry. Agents build trust through payments.
ERC-8183 — Agent Commerce
Agentic Commerce Protocol. Job escrow with confidential payments. 1% platform fee on completion. Unbypassable revenue.

Nobody else combines all four standards. This is a full-stack confidential agent commerce protocol.
